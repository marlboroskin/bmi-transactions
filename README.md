# my-transaction-tracker

Калькулятор транзакций

Простая консольная программа на Go, которая:
- Собирает транзакции (до ввода `n`)
- Выводит список
- Считает баланс
- Ждёт Enter перед выходом

Пример:

Введите транзакции (по одной). Чтобы закончить — введите 'n' Сумма: 10 Сумма: -5.5 Сумма: n

--- Транзакции ---

10.00
-5.50
Баланс: 4.50
